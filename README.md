### 说明

基于 CRA 创建，并做了定制化需求。

### 启动

```sh
yarn install

yarn start

yarn build
```

### Function:

1. Docker ci config: [Pipelines](http://10.172.94.171/frontend/xylink-cli-tpl/pipelines)
2. Jest unit test
3. Alias
4. 多环境（代理）
5. Sass
6. Support IE9-11
7. Antd 按需加载
8. webpack 优化
9. Typescript
10. Commit Hook Check
